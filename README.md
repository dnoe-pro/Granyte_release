# Granyte

> Application desktop permettant de générer des classes POO et des scripts SQL à partir d'un Modèle Conceptuel de Données (MCD) et un générateur de classe.

![Version](https://img.shields.io/github/v/release/dnoe-pro/Granyte_release?style=flat-square)
![Plateforme](https://img.shields.io/badge/plateforme-Windows-blue?style=flat-square)
![Licence](https://img.shields.io/badge/licence-MIT-green?style=flat-square)

## Installation

1. Téléchargez la dernière version depuis la page [Releases](https://github.com/dnoe-pro/Granyte_release/releases/latest)
2. Lancez l'installeur `granyte-Setup-x.x.x.exe`

> Les mises à jour seront installé automatiquement dès le lancement de l'application. 

## Fonctionnalités

### Générateur de classe
Générez automatiquement vos classes orientées objet à partir de vos attributs et les types de données.

| Langage | Statut |
|---|---|
| C# | ☑ |
| PHP | ☑|
| JavaScript | ☐ |
| Ruby | ☐ |

### Éditeur MCD
Modéliser vos données visuellement et générez vos classe POO et du scripts SQL depuis un MCD.
